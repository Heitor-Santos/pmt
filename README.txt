Esta é uma ferramenta para busca exata e aproximada de padrões em arquivos de texto.
Este projeto foi desenvolvido por Heitor da Silva Santos (hss2) e Leonardo dos Anjos Silva (las4).

Para compilar e executar a ferramenta, siga os seguintes passos:

1. Acesse o diretório /src através de linha de comando
2. Execute o comando make (isto deve criar um diretório /bin na raiz do projeto com um arquivo binário pmt dentro)
3. Acesse o diretório /bin através de linha de comando
4. Execute ./pmt -h para mais informações sobre como utilizar a ferramenta
